# 🧩 PM Checklists — Ready-to-Use

## ✔ Pre-Plan Validation
- Ask **"Why are we doing this?"** 4–7 times until value is clear.
- Kill project early if purpose is weak or not motivating.
- Bottom-up cost estimation only — no guessing.
- Use **small initial investment** until proof of viability.
- Create **Benefits Card** w/ stakeholders → What value looks like.

## ✔ Team Formation Checklist
- Dedicated time to connect as humans first.
- Goals set **with the team**, not delivered to them.
- Working agreements + behaviour standards logged & visible.
- Encourage dissent → Psychological safety is a project asset.

## ✔ WBS Development Checklist
- Gather sponsor + SMEs + stakeholders in design workshop.
- Start with simple **project concept** → visible to all.
- Generate deliverable elements individually (notes/virtual cards).
- Group → Categorise → **Decompose until ownership is clear**.
- Stop breakdown when estimate accuracy is achieved.
